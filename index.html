## Rol Actuá como un desarrollador Frontend Senior especializado en diseño temático, pixel art web y experiencias visuales inmersivas. ## Task <task> Tu objetivo es desarrollar el código completo, funcional y en un único archivo HTML5 para La Famiglia — una landing page que recopila y presenta las mejores películas del género mafioso de la historia del cine, con una estética visual inspirada en el pixel art estilo Habbo Hotel: colores oscuros con detalles neón, tipografías bold y una identidad visual que contrasta lo elegante de la mafia con lo lúdico del pixel art. Todo el texto del sitio debe estar en español con ortografía correcta, incluyendo tildes y signos de puntuación. </task> ## Design system <design_system> Aplicá estrictamente la siguiente paleta usando CSS custom properties: - --bg-main: #0d0d0d (negro profundo - fondo principal) - --bg-card: #1a1a2e (azul noche - fondo de tarjetas) - --neon-red: #ff2d55 (rojo neón - CTAs y acentos principales) - --neon-gold: #ffd700 (dorado pixel - títulos, estrellas y detalles) - --neon-cyan: #00f5ff (cyan neón - bordes activos, hover states) - --text-crisp: #f0f0f0 (texto principal) - --pixel-border: 2px solid (bordes estilo pixel, sin border-radius) Tipografía: "Press Start 2P" exclusivamente para el logo y números de ranking. Para todos los demás textos "Share Tech Mono" a un mínimo de 14px. </design_system> ## Requirements <requirements> La Landing Page debe contener las siguientes secciones en este orden exacto: 1. Header: Barra de navegación sticky con logo "La Famiglia" en "Press Start 2P". Links de navegación (Ranking, Personaje, Reseñas, Contacto). Botón CTA "Entrar a la Familia". 2. Hero Section: Escena pixel art CSS de la oficina de Vito Corleone en El Padrino (escritorio de madera oscura, sillón de respaldo alto, lámpara, cortinas pesadas, personaje mafioso sentado). Título "Las Películas que Definieron la Mafia", subtítulo y dos CTAs. 3. Sobre La Famiglia: Sección breve sobre la misión del sitio. 4. Ranking de Películas: Grid de 6 tarjetas (El Padrino, Goodfellas, Scarface, El Padrino II, Casino, Donnie Brasco), cada una con número de ranking en "Press Start 2P", personaje en pixel art CSS. 5. Creador de Personaje Mafioso: Avatar customizable con 6 atributos: - Tipo de cuerpo: Robusto, Delgado, Alto - Color de piel: Claro, Medio, Oscuro - Tipo de pelo: Corto liso, Rizado, Calvo - Accesorio de cabeza: Sin accesorio, Fedora, Borsalino negro, Gorra - Accesorio en el saco: Sin accesorio, Rosa roja - Ropa: Traje negro, Traje blanco, Campera de cuero El botón "Unirme a la Famiglia" muestra una frase aleatoria de mafia. Exportar como PNG con html2canvas (CDN). 6. Reseñas: Grid de 3 tarjetas con nombre usuario estilo Habbo, puntuación y comentario. Todas del mismo tamaño fijo. 7. Formulario de Contacto: Campos nombre, email, película favorita, mensaje. Sin backend. 8. Footer: Logo, copyright, links rápidos, íconos de redes sociales con glow neón al hover. </requirements> ## Technical constraints <technical_constraints> 1. Cero dependencias locales: Solo CDN. Sin frameworks CSS externos. 2. CSS puro embebido: Todo en un bloque <style> en el <head>. 3. JavaScript mínimo e inline: Solo para el creador de personaje, descarga con html2canvas y navegación. 4. Completitud estricta: Prohibido usar marcadores como <!-- resto del código -->. 5. Responsivo: Grid de 3 columnas en desktop, 1 columna en mobile. 6. Contenido de tarjetas sin recortes: height: auto y overflow: visible en tarjetas de Ranking. 7. Consistencia html2canvas: Construir el avatar sin pseudo-elementos ni filtros CSS complejos. </technical_constraints> ## Output format <output_format> Devolvé ÚNICAMENTE el código HTML en un solo bloque. Sin explicaciones ni conclusiones. </output_format>
